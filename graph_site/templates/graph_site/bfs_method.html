О методе обхода в ширину

Остовное дерево - ациклический связный подграф данного связного неориентированного графа, в который входят все его вершины.

Алгоритм:

  1. Добавим в очередь начальный узел
  2. Возьмем первый узел из очереди
  3. Пометим его пройденным
  4. Добавим в очередь всех его непомеченных соседей
  5. Соединим текущих соседей с предыдущим узлом последовательно
  6. Повторить пункты 2-5 пока очередь не окажется пуста

Пример работы алгоритма:

Возьмем граф из 5 узлов

Тогда работа алгоритма выглядит так

  1. Назначаем первым узел 1
  2. Берем соседи узла 1: 2, 5
  3. Добавляем их в очередь, соединяем ребрами с узлом 1, удаляем из очереди узел 1
  4. Берем соседей узла 2: 4, 3
  5. Добавляем их в очередь, соединяем ребрами с узлом 2
  6. Удаляем из очереди 2.
  7. У остальных узлов все соседи уже помеченные, потому их в очередь не добавляем. Очередь пуста
